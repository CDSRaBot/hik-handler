# Реестр файлов и библиотек проекта Hik-handler


## 1. Стек технологий и зависимости

*Данный перечень утвержден Архитектором для обеспечения безопасности и минимализма кодовой базы.*

**Стандартные библиотеки Python (Встроенные)**

pathlib: Объектно-ориентированная работа с путями.

logging: Протоколирование работы системы (режимы DEBUG/INFO).

typing: Строгая аннотация типов.

dataclasses: Контейнеры данных для SecureContext.

json/csv: Экспорт данных и работа с индексами.

**Сторонние библиотеки (Внешние)**

requests: Сетевое взаимодействие с камерами (Digest Auth).

lxml: Высокопроизводительный парсинг XML.

defusedxml: Критическая зависимость для защиты от XXE-атак.


## 2. Структура проекта @hik-handler.project

Иерархия папок на основе фактического анализа файла "Структура проекта 0.3.3.0.md".
```
@hik-handler.project/
├── app/                        # Основной пакет приложения
│   ├── communication/          # Транспортный слой (Network/Requests)
|   |   |── __init__.py
|   |   └── session.py
|   |
│   ├── configuration/          # Настройки и безопасность
│   │   ├── __init__.py
│   │   └── security.py         # Логика защиты данных
│   │   └── settings.py         # Настройки hik-handler
|   |
│   ├── engine/                 # Ядро системы (Движок)
│   │   ├── __init__.py
│   │   ├── orchestrator.py     # Оркестратор
│   │   ├── validator.py        # Валидатор XML
│   │   ├── loader.py           # Загрузчик модулей
│   │   ├── logger.py           # Обработчик логирования
│   │   └── resolver.py         # Разрешение переменных ${var}
|   |
│   └── user_interface/         # Интерфейс пользователя
│       ├── __init__.py
│       └── cli_terminal.py     # Интерактивный CLI
|
├── modules/                    # Каталог XML-описаний команд
├── docs/                       # Документация и дневники
│   └── project_registry.md     # Текущий файл (v.0.3.3.0)
├── logs/                       # Директория логов
│   └── hik-handler.log         # Основной лог-файл
├── schema/                     # Схемы валидации
│   └── module_schema.xsd       # Главная XSD-схема модулей
├── config.toml                 # Конфигурация системы
├── help.md                     # Справка по командам
├── hik-handler.py              # Точка входа (Wrapper)
├── modules.inf                 # Индекс кэшированных модулей
└── requirements.txt            # Зависимости проекта
```
