ОСНОВНЫЕ КОМАНДЫ:
  connect <IP>        Установить IP-адрес целевой камеры.
  list-modules        Показать список доступных XML-плагинов.
  run <mod> <cmd>     Выполнить действие.
                      Пример: run system get-device-info
  exit / quit         Завершить работу.

ПОДСКАЗКИ:
  - Если в конфигурации (config.toml) не указан пароль, 
    команда 'run' запросит его в защищенном режиме.
  - Используйте [TAB] для автодополнения имен команд и модулей.
  - История команд сохраняется в файле .hik_history.
